Hinweis zum Speicher

Lokale Vorschau

Hochgeladene Dateien werden hier zur Vorschau angezeigt. Da die Seite statisch auf Vercel läuft, werden Dateien aktuell nur im Browser zwischengespeichert.

Für dauerhaften Speicher

Um Bilder dauerhaft zu speichern empfehlen wir Vercel Blob Storage (kostenloser Einstieg). Anleitung unten.

Vercel Blob — Schnellstart:

  1. 1. Vercel Dashboard → Storage → Create Blob Store
  2. 2. Environment Variable BLOB_READ_WRITE_TOKEN kopieren
  3. 3. In Vercel Project Settings → Environment Variables eintragen
  4. 4. API-Route /api/upload.js für Server-Side-Upload erstellen

Einstellungen

Passwort ändern

Das Passwort ist im Quellcode der admin.html gespeichert. Suche nach ADMIN_PASSWORD und ändere den Wert.

// Zeile ca. 280 in admin.html:
const ADMIN_PASSWORD = 'esadmin2024';

Bitte ändere das Passwort vor dem ersten Einsatz!

Formspree IDs eintragen

Wenn du Formspree eingerichtet hast, ersetze in den HTML-Dateien:

index.html → YOUR_FORM_ID → z.B. xpzgkwqr
b2b.html → YOUR_B2B_FORM_ID → z.B. xbjklmno

Medien löschen

Alle lokal gespeicherten Medien aus dem Browser entfernen.
